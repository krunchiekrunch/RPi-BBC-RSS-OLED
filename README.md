# RPi-BBC-RSS-OLED
Display BBC headlines on a SSD1306 OLED display on a Raspberry Pi

# Setup

Connect a 128x64 SSD1306 I2C OLED display to the Pi

Enter python3 virtual enviroment

```
pip install requests Pillow adafruit-blinka adafruit-circuitpython-ssd1306
python main.py
```

Based on [ukkokalevala/RSS_BBC_Head_Lines](https://github.com/ukkokalevala/RSS_BBC_Head_Lines)
